1. Что понимается под гетерогенной параллелизацией?
Гетерогенная параллелизация — это подход в параллельных вычислениях, при котором задачи распределяются между различными типами вычислительных устройств с разными архитектурами (например, центральные процессоры (CPU), графические процессоры (GPU). 
2. В чём принципиальные различия архитектур CPU и GPU?
CPU — это "управляющий центр", способный выполнять сложные логические операции и координировать работу всей системы, в то время как GPU — это "вычислительный ускоритель", который позволяет ускорить массово-параллельные операции, такие как обработка изображений или векторные вычисления.
3. Какие типы задач лучше подходят для выполнения на GPU, а какие — на CPU?
Cложные вычисления могут выполняться на GPU, а задачи управления — на CPU
